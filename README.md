# Svelte & Snowpack template

```bash
## Install template
$ npx degit uujei/svelte-snowpack-template your_project_name

## Change directory
$ cd your_project_name

## Install dependencies
$ npm i

## Start dev server
$ npm run dev
```

- Svelte
- Snowpack
- TypeScript
- SCSS
- Autoprefixer/PostCSS
- Web test runner
- Chai
- Reset.css
